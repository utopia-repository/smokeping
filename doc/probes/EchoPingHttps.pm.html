NAME

probes::EchoPingHttps - an echoping(1) probe for SmokePing

OVERVIEW

Measures HTTPS (HTTP over SSL) roundtrip times (web servers and caches) for SmokePing.

SYNOPSYS

 *** Probes ***
 + EchoPingHttps
 binary = /usr/bin/echoping # mandatory
 *** Targets ***
 probe = EchoPingHttps
 + PROBE_CONF
 url = / 
 ignore-cache = yes
 force-revalidate = no
 port = 443 # default value anyway

DESCRIPTION

As EchoPingHttp(3pm), but SSL-enabled.
