NAME

probes::EchoPingSmtp - an echoping(1) probe for SmokePing

OVERVIEW

Measures SMTP roundtrip times (mail servers) for SmokePing.

SYNOPSYS

 *** Probes ***
 + EchoPingSmtp
 binary = /usr/bin/echoping # mandatory
 *** Targets ***
 probe = EchoPingSmtp

DESCRIPTION

Supported probe-specific variables: those specified in EchoPing(3pm) documentation.

Supported target-specific variables: those specified in EchoPing(3pm) documentation except fill, size and udp.
